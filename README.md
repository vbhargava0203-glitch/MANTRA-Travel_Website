# MANTRA 2026 Sample Travel Website

This is a beginner-friendly sample frontend website for the MANTRA 2026 Summer School assignment.

## Website Type
Travel Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included
- Home
- About
- Destinations
- Packages
- Gallery
- Booking
- Contact

## 📁 Project Structure
MANTRA Travel_Website/
├── index.html          # Home page
├── about.html          # About Us page
├── destinations.html   # Destinations page
├── packages.html       # Tour Packages page
├── gallery.html        # Photo Gallery page
├── contact.html        # Contact Us page
├── booking.html        # Tour Booking / Enquiry page
├── style.css           # All styles (responsive + theme)
├── script.js           # JavaScript interactivity
├── images/
│   ├── hero.svg
│   ├── goa.svg
│   ├── kerala.svg
│   ├── jaipur.svg
│   ├── shimla.svg
└── README.md
```
## Features
- Responsive design for mobile, tablet and desktop
- SEO tags on every page
- Image alt text
- Mobile navigation menu
- Booking form validation using JavaScript
- Package cost calculator using JavaScript
- Gallery filter using JavaScript

## How to Run
Open `index.html` in any browser.
